The Challenge

Currently, invoice activity code assignment is a manual, time-intensive process that takes several months to complete. With 3,768 invoice lines per month and 81% missing activity codes (3,057 invoices), the finance team faces:

  • Incomplete financial reporting and delayed spend analytics
  • Inability to track IT vs. non-IT spend in real-time
  • High manual effort reviewing supplier names and commodity codes
  • Inconsistent classification decisions across team members
  • 938 unique suppliers requiring ongoing classification

Current vs. Proposed Process

❌ Current Manual Process

  • Manually review each invoice line
  • Research supplier business type
  • Cross-reference commodity codes
  • Look up historical assignments (Excel VLOOKUP)
  • Takes several months per quarter
  • Inconsistent decisions across reviewers
  • No learning from corrections
  • Difficult to audit classification rationale

✅ Automated Classification System

  • Instant classification (hours not months)
  • AI-powered fuzzy supplier matching
  • Automated keyword analysis (commodity + description)
  • Database-backed historical pattern learning
  • Confidence scoring (auto-assign ≥70%)
  • Consistent rule-based decisions
  • Learns from manual corrections (feedback loop)
  • Complete audit trail with rationale

Key Capabilities

🎯 Intelligent Matching

Fuzzy supplier name matching handles variations while preserving important suffixes. Prioritizes supplier → commodity → description → historical patterns.

📊 Confidence Scoring

Three-tier system: Auto-assign (≥70%), Suggest alternatives (20-70%), Manual review (<20%). Ensures quality while maximizing automation.

🔍 Smart Detection

Automatically identifies non-IT vendors (cleaning, benefits, legal) and flags for admin approval with suggested codes.

🔄 Learning System

Feedback loop captures manual corrections and updates classification rules. System improves accuracy over time.

📋 Audit Trail

Complete history of classifications, confidence scores, methods used, and manual overrides for compliance and review.

⚡ Batch Processing

Process all 4 months of historical data (15,072 invoice lines) in minutes. Export review reports for validation.

Expected ROI

Reduce classification time from months to hours (95%+ time savings)

Auto-assign 60%+ of invoices with 95%+ accuracy

Enable real-time spend analytics and financial reporting

Implementation Timeline

Phase 3: Batch Processor

Phase 4: Testing & Iteration

Review auto-assignments, validate admin alerts, fine-tune thresholds, add keyword rules.
